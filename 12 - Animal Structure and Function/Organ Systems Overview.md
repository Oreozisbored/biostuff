# Organ Systems Overview

## Overview
Key organ systems work together to maintain [[Homeostasis]]:
- Circulatory (transport)
- Respiratory (gas exchange)
- Digestive (food breakdown)
- Nervous (control via [[Neurons]])
- Endocrine (hormones)
- Immune/Lymphatic (defense)
- Excretory (waste removal)
