# Base Pairing Rules

## Overview
Also known as Chargaff's rules.

In DNA:
- Adenine (A) always pairs with Thymine (T) via 2 hydrogen bonds.
- Guanine (G) always pairs with Cytosine (C) via 3 hydrogen bonds.

In RNA, Uracil (U) replaces Thymine. This complementary pairing is the basis for [[DNA Replication]] and [[Transcription]].
