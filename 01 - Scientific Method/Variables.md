# Variables

## Overview
Variables are factors that can change in an [[Experiment]]. 

## Types of Variables
- **Independent Variable**: The factor that is deliberately changed by the scientist.
- **Dependent Variable**: The factor that is measured or observed as a result.
- **Controlled Variables (Constants)**: Factors kept exactly the same across all groups in a [[Controlled Experiment]].
