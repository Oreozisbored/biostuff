# Diploid and Haploid

## Overview
- **Diploid (2n)**: A cell containing two sets of chromosomes, one set inherited from each parent. (e.g., human [[Somatic Cells]] have 46).
- **Haploid (n)**: A cell containing only one set of chromosomes. (e.g., human [[Gametes]] have 23).

[[Meiosis]] reduces ploidy from diploid to haploid. Fertilization restores it.
