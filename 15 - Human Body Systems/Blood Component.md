# Blood Component

## Overview
Blood is a specialized connective tissue.

Consists of:
- **Plasma**: Liquid matrix containing water, ions, proteins.
- **Red Blood Cells (Erythrocytes)**: Carry oxygen using hemoglobin.
- **White Blood Cells (Leukocytes)**: Part of the [[Immune System]].
- **Platelets**: Cell fragments involved in clotting.
